EVA2Z Connect - GPS Tracking for Electric Vehicles

2023 - 2024 Evatoz Solutions Pvt. Ltd. Team: 3 Developers

Project Overview

EVA2Z Connect is a comprehensive GPS tracking application designed specifically for electric vehicles. Built using Kotlin Multiplatform (KMP), it provides real-time vehicle tracking, route history, geofencing, and device management across both Android and iOS platforms with 85% code sharing.

Purpose & Problem Statement

Problem

EV owners lacked a unified solution to monitor vehicle location, battery status, and trip analytics in real-time.

Solution

Cross-platform GPS tracking app with live monitoring, intelligent geofencing, and comprehensive analytics dashboard.

Technology Stack

Kotlin Multiplatform (KMP) Jetpack Compose SwiftUI Google Maps SDK Firebase Firestore Firebase Analytics MVVM Architecture Coroutines & Flow Ktor Client SQLDelight

Development Lifecycle

  • Defined core features: real-time tracking, geofencing, device management

2

Core Development (Week 3-10)

  • Implemented shared networking layer with Ktor
  • Built GPS tracking service with background location updates
  • Developed real-time map interface with Google Maps SDK
  • Created Firebase integration for device data sync
  • Implemented geofencing with customizable boundaries
  • Built device management: add, remove, configure devices
3

Platform-Specific UI (Week 11-14)

  • Android: Jetpack Compose for Material Design 3
  • iOS: SwiftUI for native iOS experience
  • Implemented auto-zoom and location tracking features
  • Created responsive layouts for tablets and phones

Testing & Optimization (Week 15-17)

  • Unit testing for shared business logic
  • UI testing on multiple devices and screen sizes
  • Performance optimization: reduced battery drain by 30%
  • Fixed BLE connection stability issues
5

Deployment & Monitoring (Week 18-20)

  • Google Play Store and App Store submission
  • Firebase Analytics integration for user behavior tracking
  • Crash reporting with Firebase Crashlytics
  • Post-launch bug fixes and improvements

Niharika Attendance System - AI-Powered Face Recognition

2024 Evatoz Solutions Pvt. Ltd. Solo Project

Project Overview

An intelligent attendance management system leveraging on-device ML Kit for face recognition. Successfully transitioned from expensive cloud-based facial recognition to a free, on-device solution while achieving 99.65% accuracy and passing Google Play Store production approval.

Purpose & Problem Statement

Problem

Manual attendance tracking was time-consuming and prone to proxy attendance. Cloud-based facial recognition was costing ₹3,290/month.

Solution

Built an on-device ML-powered system that eliminated cloud costs while maintaining 99.65% accuracy with real-time face detection.

Technology Stack

Android (Kotlin) Jetpack Compose ML Kit Face Detection CameraX Firebase Firestore Firebase Storage Room Database MVVM Architecture Coroutines & Flow Hilt DI

Development Lifecycle

1

Research & Planning (Week 1-2)

  • Evaluated cloud vs. on-device ML solutions
  • Tested ML Kit face detection accuracy and performance
  • Designed face embedding storage and matching algorithm
  • Created database schema for attendance records
2

Core ML Integration (Week 3-6)

  • Integrated ML Kit Face Detection API
  • Built face enrollment system with multiple angle capture
  • Implemented face matching with cosine similarity algorithm
  • Optimized face detection for different lighting conditions
  • Added liveness detection to prevent spoofing
3

UI Development (Week 7-9)

  • Created camera preview with face detection overlay
  • Built admin panel for employee management
  • Designed attendance dashboard with analytics
  • Implemented role-based access control
4

Testing & Accuracy Tuning (Week 10-12)

  • Tested with 100+ employee faces in various conditions
  • Achieved 99.65% accuracy through threshold optimization
  • Reduced false positives to less than 0.5%
  • Performance testing on low-end devices
5

Deployment (Week 13-14)

  • Google Play Store production approval
  • Set up Firebase for cloud sync and backup
  • Created admin documentation and training materials
  • Deployed to company with 50+ employees
